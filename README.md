## About

KTed is an engaging game about exploring the history of Kazakhstan, available in three languages (English, Russian and Kazakh). Its interactive approach to learning allows students to immerse themselves in a virtual world, where they can interact with historical figures from Kazakhstan's past and discover various real-world sources scattered throughout the game world
